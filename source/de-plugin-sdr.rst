==========
SDR Plugin
==========

The SDR (Software Defined Radio) Plugin integrates radio communication capabilities with DroneEngage, enabling spectrum monitoring and radio-based applications.

.. youtube:: _Ek24PA5dso

|

Features
========

- **Spectrum Monitoring** - Monitor radio frequencies
- **Signal Detection** - Detect and locate radio signals
- **Remote Radio Control** - Control SDR hardware from the web client

|

Hardware Requirements
=====================

- Compatible SDR hardware (RTL-SDR, HackRF, etc.)
- USB connection to Raspberry Pi

|

Configuration
=============

The SDR plugin configuration is stored in ``de_sdr.config.module.json``.

|

   This is an advanced plugin. See :ref:`de-dev-plugin` for development and integration details.
