===============
Getting Started
===============

This guide walks you through the essential steps to get DroneEngage up and running on your drone. Follow these steps in order for the best experience.

.. youtube:: Q-OyRnisq8U

|

Quick Start Overview
====================

Getting started with DroneEngage involves three main steps:

1. **Select Your Hardware** - Choose the right Raspberry Pi and connectivity options for your needs
2. **Create Your Account** - Register and obtain your Access Code from the DroneEngage cloud
3. **Install DroneEngage** - Set up the software on your Raspberry Pi

.. tip::
   For the fastest setup, use the **Ready-Made RPI Image** which comes pre-configured with all DroneEngage modules.

What's Next?
============

After completing the installation:

- **Configure your unit** using the :ref:`de-rpi-image-tools` web interface
- **Connect to the Web Client** at `https://droneengage.com:8021/webclient.html <https://droneengage.com:8021/webclient.html>`_
- **Test with SITL** before flying - see :ref:`de-simulators`
